# E-Commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data using Excel, SQL, and Power BI to identify sales trends, profitability, regional performance, and top-performing products.

---

## Tools Used
- Excel
- MySQL
- Power BI

---

## Project Workflow
1. Cleaned and transformed raw dataset using Excel.
2. Imported cleaned CSV data into MySQL.
3. Performed SQL analysis using aggregation and grouping queries.
4. Built an interactive Power BI dashboard for business insights.

---

## Dashboard Features
- Key Metrics Overview
- Monthly Sales Trend
- Regional Sales Analysis
- Profit by Category
- Top 10 Products
- Interactive Filters (Slicers)

---

## Key Insights
- West region generated highest sales.
- Technology category produced highest profit.
- Some products showed negative profitability despite strong sales.
